## 📊 Análisis de Complejidad
Basado en la teoría de la materia, el rendimiento de este algoritmo es:
* **Mejor caso / Promedio:** $O(n \log n)$
* **Peor caso:** $O(n^2)$ (cuando el pivote es el menor o mayor elemento)
* **Espacio en memoria:** $O(\log n)$ debido a la recursividad.
